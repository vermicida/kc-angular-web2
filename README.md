## Getting started

Para ejecutarlo, basta con clonar este repositorio y ejecutar lo siguiente:

Para instalar dependencias:

```bash
$ npm install
```

Para ejecutar la app en entorno de desarrollo:

```bash
$ ng serve
```

Solo queda ir `http://localhost:4200/` en un navegador web :-)
